# 🎓 GPA & CGPA Calculator

This project is a Python-based application that calculates both **GPA (Grade Point Average)** and **CGPA (Cumulative Grade Point Average)** using user-input grades and subject credits.

## ✨ Features

* 📊 Calculate GPA for a single semester
* 🎯 Calculate CGPA across multiple semesters
* 🔢 Grade-to-point conversion system
* ⚡ Simple and efficient calculation logic
* 🧮 Works inside Jupyter Notebook

## 🛠️ Technologies Used

* Python 🐍
* Jupyter Notebook

## 📈 Functionality

* Accepts grades (A, B, C, etc.)
* Accepts subject credits
* Computes GPA using weighted average
* Computes CGPA by combining multiple GPA values

## 🚀 Use Case

Useful for students to track and evaluate their academic performance easily.
